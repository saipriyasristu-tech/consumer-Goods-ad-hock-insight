
# 📊 Consumer Goods Ad-Hoc Insights
SQL-based analysis of consumer goods data to generate business insights using Excel and Power BI dashboards.

## 🔹 Project Overview
This project analyzes consumer goods data using SQL to solve 10 business problems and generate meaningful insights. The goal is to understand product performance, sales trends, and customer behavior to support data-driven decision-making.

## 🔹 Objectives
- Perform ad-hoc analysis using SQL queries  
- Identify key business trends and patterns  
- Present insights using Excel and Power BI dashboards  

## 🔹 Tools & Technologies
- SQL (MySQL)  
- Excel (Data visualization)  
- Power BI (Interactive dashboards)  

## 🔹 Key Insights
- Unique product count increased by *36% in 2021*, showing business expansion  
- Network and Accessories segments have the highest product count  
- Accessories segment showed significant growth compared to other segments  
- Sales show seasonal trends with peaks towards the end of the year (especially November)  
- Higher discounts are offered to key customers  

## 🔹 Recommendations
- Focus on high-growth segments like Accessories  
- Optimize discount strategies to improve profit margins  
- Plan inventory based on seasonal demand trends  
- Strengthen performance in lower-performing segments
